# order-form-ui

A React UI for calculating optimal pack fulfillment for product orders.

## Prerequisites

- Node.js (v18 or higher recommended)
- npm

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open your browser at [http://localhost:5173](http://localhost:5173)

## API Backend

This UI expects a backend API running at `localhost:8080`. The app proxies requests to `/api/*` endpoints. See `swagger.json` for the full API specification.
